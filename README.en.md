# dockit4j-ui

#### Description
dockit4j-ui 是 Dockit4j 的前端子项目，提供基于 OpenAPI 3 规范的现代化 API 文档界面。它旨在替代传统 Swagger UI，采用更清晰的交互设计与更强的自定义能力，支持多种主题、分组视图、接口调试等高级功能，带来更优的开发者体验。
